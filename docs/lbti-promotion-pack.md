LBTI 是一个面向移动端传播的恋爱人格测试：通过真实关系场景题，测出你在恋爱里的关系角色，并生成可保存转发的专属结果图。

## 二、主推卖点

### 1. 不是普通 MBTI 换皮

LBTI 不是把 MBTI 的四个字母简单换成恋爱标签，而是围绕恋爱关系里的真实行为设计：消息回应、确认需求、边界感、冲突方式、关系推进、自我保护等维度都会影响结果。

### 2. 16 种恋爱角色，更适合截图传播

每个结果不是学术名词，而是更适合社交平台传播的角色名称，比如“消息免打扰体”“试用期人”“舔者”“纯爱战神”等。用户可以快速用一句话描述自己，也方便朋友之间互相调侃和转发。

### 3. 每个角色有三张脸

同一个角色会拆成三种展示面：

- 自嘲面：适合发朋友圈、群聊、评论区自黑。
- 动物面：更可爱，更适合头像、贴纸和女性向传播。
- 甜心面：更柔和，适合情绪价值和关系解释。

### 4. 结果页可直接保存分享图

测试完成后会弹出一张结果卡片，用户可以直接保存图片并转发。分享图包含角色图、角色名、结果信息和站点链接，适合公众号文章、小红书封面、朋友圈二次传播。

### 标题候选

- 16 种恋爱角色：你是甜心、刺猬，还是已读不回成精？
- 一个专门测恋爱行为的 LBTI，建议和对象一起玩
- 别只测 MBTI 了，来测测你在恋爱里到底是什么角色

### 开头文案

很多人格测试都在回答“你是谁”，但恋爱里更有意思的问题其实是：你靠近一个人时会变成什么样。

有人平时很正常，一进入关系就开始反复确认；有人嘴上说无所谓，心里已经把聊天记录复盘三遍；有人看起来很甜，其实边界感强到像自带门禁；还有人永远慢半拍，但一认真起来又让人很难招架。

所以我做了一个 LBTI 恋爱人格测试。它不试图给你下诊断，只是把你在关系里的习惯、反应、防御和表达方式，压缩成一个更好懂、更好笑、也更容易传播的恋爱角色。

### 正文结构

1. LBTI 是什么：Love Behavior Type Indicator，恋爱行为类型测试。
2. 它测什么：不是你适合谁，而是你在关系里如何回应、靠近、退缩、表达和保护自己。
3. 为什么是 16 个角色：为了让结果既有区分度，又能被用户一眼记住。
4. 为什么有三张脸：自嘲面适合传播，动物面适合可爱表达，甜心面适合柔和解释。
5. 怎么玩：打开官网，点击开始测试，完成答题后保存结果图。
6. 适合谁玩：情侣、暧昧对象、闺蜜群、大学生、MBTI 爱好者。

- 最重要的转化动作是：让用户完成测试并保存结果图。
