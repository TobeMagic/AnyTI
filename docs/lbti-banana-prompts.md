# LBTI Banana 文生图 Prompt 包（v6）

> **核心变化 v6**：完全基于现有的扁平几何插画 SVG 风格。现有角色是 flat illustration style（非3D渲染），特征：扁平六边形头、polygon折面、暖棕系色板、#4A3828深棕描边。

---

## 风格锁定（必须全部包含）

每条 prompt 必须包含以下关键词，精确匹配现有 SVG 角色风格：

```
flat 2D illustration, flat color-block, no depth, no 3D, no lighting,
no shading, no gradients, no soft shadows, pure polygon geometry,
solid flat colors, clean outlines, crisp edges, no blur,
faceted geometric mascot, brand mascot style
```

---

## 风格色板（必须匹配）

```
skin tone: #D4C5B2, #C4B49E, #EAE0D4
outline: #4A3828 (dark warm brown, 1.5-2px)
hair/dark features: #5C4A38, #8B6F4E
body: #C8B89C, #9E8E78
leg: #9E8E78, #78A068
background: #F5F0EB (warm off-white)

```
